1. MainScene (담당: 김재영)
메인화면에서 시작 버튼 누르면 컷씬으로 변환 컷씬에서 간단한 스토리와 튜토리얼 내용 보여주고 stagepanel로 변환
stagepanel에서는 지도위에 버튼을 놓아서 그 스테이지를 들어갈수있게 하는 기능, 각 스테이지마다 배경값과 난이도를 
변경할수있는 기능 추가 (예를들어 백년관 버튼누르면 gamescene이 백년관 배경과 기존에 설정한 적 배치등을 백년관 스테이지에 맞게 할수있게 기능추가) , 엔딩 조건 (모든 스테이지깨기) 확인해서 마지막 endingpanel 로 변환(원래 없었는데 추가)
추가적으로 기본적인 UI 들 제작(stagePanel, gameScene)

+커밋내용 추가
현재 올려놓은 이미지들은 개발상 편의를 위해, 그리고 이해를 돕기위해 넣어둔 샘플들입니다. 기능자체만 먼저 보시면 됩니다.

stagePanel gameScene 뒷배경하고 컴포넌트들은 나중에 이미지만 바꾸면 되게 구성완료. 각 스테이지마다 배경하고 건물(기지) 다르게 할수있도록 구현.
gamescene 카메라무빙 추가 (방향키 wasd 마우스). 엔딩조건 확인해서 마지막 endPanel로 가는 기능도 구현.


2. GameScene (담당:송수한)
각 캐릭터들을 디자인할수 있는 베이스를 생성. 예를들어 샘플 이미지등을 놓고 캐릭터가 움직일때, 공격할때 죽을때 등 모션을 구현할 수 있는 템플릿을 제작. 또한 실제로 캐릭터들끼리 전투를 할때 필요한 스크립트들 제작 (예를들어 캐릭터들끼리 전투를 해서 공격하거나 피격당하거나 했을때 hp 체크하고 hp가 0되면 죽고 그런 기초적인 기능들)

https://www.youtube.com/watch?v=EZ-6pQs-tB8 이런느낌 (포탑 부분 뺴고)

3. upgradeScene (담당:오세완) (gamescene 에서 캐릭터 소환하는데 드는 비용을 마나, upgradescene 에서 캐릭터들 영구
적으로 업그레이드 하는 비용을 골드라고 일단 가정.)

캐릭터들을 업그레이드 할 수 있는 기본적인 UI 만들기. (실질적으로 업그레이드를 하는 기능은 추후에 추가) 추가적으로
전체적인 게임밸런싱 구상. 캐릭터들 기본 능력치 (체력, 방어력(필요한지 논의필요), 공격력, 공격사거리 등), 캐릭터들 소환할때 드는 마나, 그에 따른 마나 획득량 (팔라독 같은경우는 기본 마나 수급량만 존재하고. 전쟁시대는 기본적인 마나 수입에 적 처치할때마다 그에 상응하는 마나 획득), 스테이지 클리어시 골드 획득량과 업그레이드 하는데 드는 골드 비용 (현재 각 캐릭터 마다 기본 1단계에서 시작해서 4단계까지 총 3업을 할수있도록 기획함), 업그레이드 마다 올라가는 능력치 와 얼마나 올라가는지 , 각 기지 체력 등. 그리고 각 스테이지 마다 적 나오는 걸 배치를 어떻게 할껀지 (초기 논의 필요).

실질적으로 당장 기능을 구현해야 하는건 업그레이드 UI, 마나와 골드 획득 and 소비 기능. 
나머지는 추후에 필요한 기능을 위해 미리 구상만 하기(캐릭터 능력치, 업그레이드 기능 등).
